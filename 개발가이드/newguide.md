네, **"한 사람과 여러 개의 채팅방(주제별 대화방)을 가질 수 있다"**는 점을 반영하여 디자인을 개선해 보겠습니다.

보통의 메신저는 친구 한 명당 하나의 1:1 채팅방만 존재하지만, 말씀하신 대로 "친구 A와 여행 계획 방", "친구 A와 코딩 공부 방" 처럼 주제별로 여러 방을 만들 수 있다면 훨씬 유용합니다.

이를 위해 친구 목록(Friends) 탭의 디자인과 기능을 대폭 강화했습니다.

🔍 주요 변경 및 추가 기능
친구 목록 확장 (Accordion UI):

친구를 클릭하면 바로 채팅방으로 이동하는 대신, 해당 친구와 관련된 모든 채팅방 목록이 아래로 펼쳐집니다.

예: '김코딩' 클릭 -> [ 💬 1:1 대화 ], [ ✈️ 일본 여행 팟 ], [ 💻 코딩 스터디 ] 목록 노출.

주제별 새 채팅방 만들기:

친구 카드 안에 + 새 주제로 대화하기 버튼을 추가했습니다.

이 버튼을 통해 같은 친구와도 새로운 주제의 방을 계속 만들 수 있습니다.

직관적인 아이콘:

일반 1:1 대화방과 주제가 있는 대화방을 아이콘으로 구분하여 보여줍니다.
